# MED 2 - Conjunto de Medidas Definido
**Projeto:** Colorindo o Daltonismo

## 1. Definição das Métricas
Abaixo estão as medidas selecionadas para atender aos objetivos definidos no MED 1:

| ID | Medida | Unidade | Descrição |
| :--- | :--- | :--- | :--- |
| **M01** | Cobertura de Código | Percentual (%) | Proporção de linhas executadas pelo Pytest em relação ao total de linhas válidas. |
| **M02** | Volume de Commits | Inteiro | Total de submissões realizadas no repositório para medir a cadência de trabalho. |
| **M03** | Issues Resolvidas | Inteiro | Total de requisitos ou bugs (issues) concluídos com sucesso. |
| **M04** | Execuções de Workflow| Inteiro | Quantidade de vezes que o pipeline de CI/CD foi disparado no GitHub Actions. |
| **M05** | Colaboração (PRs) | Inteiro | Quantidade de Pull Requests criados para revisão de código por pares. |
