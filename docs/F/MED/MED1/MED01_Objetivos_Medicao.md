# MED 1 - Necessidades de Informação e Objetivos de Medição
**Projeto:** Colorindo o Daltonismo
**Data:** 21/01/2026

## 1. Necessidades de Informação
Para garantir a maturidade do projeto no Nível F do MPS-BR-SW, a equipe identificou as seguintes necessidades:
* **Confiabilidade Algorítmica:** Validar se o motor de identificação de cores é preciso o suficiente para não induzir o usuário ao erro.
* **Estabilidade do Processo:** Verificar se o ritmo de desenvolvimento e a integração contínua estão protegendo o produto de regressões.

## 2. Objetivos de Medição
* **Objetivo 1:** Assegurar que 100% da lógica crítica de cálculo matemático e API esteja coberta por testes automatizados.
* **Objetivo 2:** Monitorar o engajamento e a granularidade do desenvolvimento através do histórico de versionamento.
* **Objetivo 3:** Validar a saúde da integração contínua frente ao volume de modificações.
