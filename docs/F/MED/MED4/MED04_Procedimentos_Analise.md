# MED 4 - Procedimentos para Análise e Comunicação
**Projeto:** Colorindo o Daltonismo

## 1. Critérios de Análise
* **Meta de Cobertura:** O projeto deve manter uma cobertura global superior a 90%.
* **Meta de Integração:** Pelo menos 2 workflows de CI devem ser executados com sucesso para cada issue resolvida para garantir estabilidade.

## 2. Comunicação de Resultados
* Os desvios identificados (ex: queda na cobertura) são comunicados através de Issues técnicas de "Refatoração".
* O relatório final de medição é distribuído para os líderes de projeto e serve de evidência para a auditoria de processos.
